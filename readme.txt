说明：

本工具需要在WINDOWS环境下运行，并且保证已成功安装JAVA环境。

本工具的使用流程如下：
1：将所有待标注的EXCEL文件复制到“1-全部文件”下
2：运行“step1-抽取待标注文件.bat”，将设定比例的文件复制到“2-人工标注文件”下
3：人工对“2-人工标注文件”下的所有文件进行标注，并保存
4: 运行“step2-训练标签库.bat”，将“2-人工标注文件”下完成标注的文件吸取到标签库
5：运行“step3-自动标注所有文件.bat”，将“1-全部文件”下的所有文件标注并另存为到“3-完成自动标注的全部文件”下
6：运行“step4-抽取待检查文件.bat”，将设定比例的文件从“3-完成自动标注的全部文件”下复制到“4-人工抽检文件”下
7：人工对“4-人工抽检文件”下的文件进行检查，并纠正发现错误，若有误，跳到第4步
8：结束



标签库的编辑可在“标签库.xlsx”中进行，注意保持格式一致
项目配置信息存储于config.ini中
词向量数据存储于vectors.bin中
